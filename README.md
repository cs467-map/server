A Node.js/Express powered server for all your mapping needs!

# Quickstart

```sh
npm install
npm run build
npm run start
```

# Database Setup

## Install PostgreSQL

[OS-specific instructions](https://www.postgresql.org/download/).

```sh
# crate a database user
createuser --interactive

# create a database
createdb COOLDATABASE

# access database
psql -d COOLDATABASE
```
